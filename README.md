# Airbnb-Wanderlust 🏡

A full-stack Airbnb-inspired web application where users can create, view, edit, and delete property listings.

## 🚀 Features
- Create new listings
- View all listings
- Edit listings
- Delete listings
- RESTful routing
- MongoDB database integration

## 🛠️ Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS
- CSS

## 📁 Project Setup

1. Clone the repository
2. Install dependencies:
   npm install
3. Start server:
   node app.js
4. Open in browser:
   http://localhost:8080/listings
